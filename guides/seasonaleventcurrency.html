Event Shop Spending Guide

Updated July 2026

Introduction

Most seasonal events feature multiple currencies with different shops and rewards. This guide explains how to prioritize your purchases so you get the highest long-term value, especially as a free-to-play player.

Event Currencies

🔹 Common Currency

  • Daily Check-in
  • Event Roulette
  • Dummy Boss Damage Rewards

🔸 Silver Currency

  • Event Activity

⭐ Gold Currency

  • Event Activity
  • Ranking Rewards

Common Currency Priority

  1. Legendary Event Equipment (Weapon / Back Accessory)
    • Highest priority.
    • Upgrade it to 8★ before buying anything else.
  2. Stellar Fragment & Meteoric Steel
  3. Emblem Dice & Emblem Shards
  4. Runes
    • New players: Buy your 2–3 highest-level rune types.
    • Endgame players: Usually safe to skip.
  5. Rush Event Resources
    • Relic Shard
    • Mystic Ore
    • Guardian Stone Fragment
  6. Optional
    • Goblin Cards
    • Orbs (10/10)
    • Moonstones

Silver Currency Priority

  1. Event Entry Tickets

    Example: Premium Summer Fishing Bait or similar items used to enter the event activity.

  2. Dungeon Keys
    • Rune Keys
    • Relic Keys
    • Accessory Keys
  3. Emblem Essence

If you still have Silver Currency remaining, purchase whichever upgrade materials you currently need most.

Gold Currency Priority

⚠️ Save your Gold Currency until the event is almost over.

Ranking rewards often grant additional Gold Currency, allowing you to determine your final budget before making purchases.

  1. Mythic Divine Weapon
    • Skip this if you've already obtained one from the event.
  2. Mythic Pet
    • Skip this if you've already obtained one from the event.

If you have extra Gold Currency, prioritize:

  • Stellar Fragment
  • Meteoric Steel
  • Mythic Trace
  • Legendary Emblem Dice
  • Profile Frame (Cosmetic)

General Tip

Always prioritize limited-time event-exclusive rewards before purchasing upgrade materials.

Most upgrade resources can be obtained later through normal gameplay, while event-exclusive items may not return for a long time.

Example (Current Event)

At the time of writing, the Midsummer Vacation event uses:

  • 🔹 Beach Parasol — Common Currency
  • 🔸 Summer Fishing Coin — Silver Currency
  • ⭐ Golden Summer Fishing Coin — Gold Currency

Activity Upgrade Tip

Most seasonal events include an upgrade system for their activity (for example, the Fishing Rod during the Midsummer Vacation event). Upgrading it increases your chances of obtaining higher-tier rewards, including Gold Currency.

Based on previous events, you can usually fully upgrade the activity while still having enough Silver Currency to purchase:

  • Premium Activity Tickets
  • Rune Keys
  • Relic Keys
  • Accessory Keys

⚠️ Spend wisely. Event rewards and Gold Currency income are still RNG-based, so your final earnings may vary. If you're uncertain, consider waiting a few days before spending all of your Silver and Gold Currency.
