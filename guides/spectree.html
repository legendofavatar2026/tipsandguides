Specialization Tree Guide ⚔️

Updated July 2026

Introduction

The Specialization Tree allows you to customize your build by unlocking powerful passive effects. Choosing the correct specialization greatly improves your performance depending on the content you're tackling.

Different builds require different specialization paths, so switching between presets is highly recommended.

Recommended Specialization Paths

Build Recommended Line
Double Combo ⛈️ Storm
Triple Combo 🌪️ Typhoon
Counter (PvP) 🪨 Iron Wall
Counter (PvE) 🏰 Fortress
Skill Critical Damage (PvP) ☄️ Collapse
Skill Critical Damage (PvE) 💥 Shatter

Emblem Compatibility

Some 6/6 Emblem Sets activate effects when using a specific skill type. Each Specialization Tree belongs to one of the following categories.

Skill Type Specialization Lines
☢️ Corrosion Shatter • Collapse
🌪️ Wind Storm • Typhoon
🪨 Stone Iron Wall • Fortress

Preset Recommendation

Instead of resetting your specialization every time you switch content, make use of the preset system.

Create separate presets for Combo, Counter, Skill, World Boss, Arena, and other activities. This allows you to swap builds instantly without resetting your Specialization Tree.

Quick Summary

  • 🌪️ Typhoon — Triple Combo Builds
  • ⛈️ Storm — Double Combo Builds
  • 🏰 Fortress — PvE Counter Builds
  • 🪨 Iron Wall — PvP Counter Builds
  • 💥 Shatter — PvE Skill Critical Builds
  • ☄️ Collapse — PvP Skill Critical Builds
  • 💾 Save multiple presets to switch builds quickly.
