Triple Combo Chance (TCC) Guide 💥💥💥

Updated July 2026

Introduction

Triple Combo Chance (TCC) is divided into two separate categories: Base TCC and Buff TCC. Understanding how these two categories interact is important because they do not simply add together.

Instead, Base TCC provides the foundation for triggering Triple Combos, while Buff TCC acts as a multiplier that increases your existing Base TCC.

Base Triple Combo Chance

  • Naktia (Mythic) — 10%
  • T7 Typhon Specialization — 20%
  • Lv.40 Circle Emblem — 10% (Unconfirmed)

Total Base TCC

40%

Buff Triple Combo Chance

  • Naktia Transcendence Wing — 4% × 5 = 20%
  • Tabby — 40%
  • Prominence — 30%

Total Buff TCC

90%

Calculation Formula

Base TCC and Buff TCC belong to different categories.

Base TCC × (1 + Buff TCC)

Example

40% × (1 + 90%) =

76% Triple Combo Chance

This means the current maximum achievable Triple Combo Chance is 76%, not 130%.

Proof

Buff TCC cannot generate Triple Combos by itself.

A simple test is using Draco while having 0% Base TCC. Even with Draco's Triple Combo buff active, you'll only observe DCD sequences and never TCD sequences.

This demonstrates that Buff TCC only amplifies existing Base TCC and cannot create Triple Combo Chance on its own.

Summary

  • ✅ Base TCC and Buff TCC are separate categories.
  • ✅ Values within the same category are added together.
  • ✅ Different categories multiply with each other.
  • ✅ Current maximum TCC is 76%.
  • ✅ Buff TCC requires Base TCC to function.
